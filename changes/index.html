What's new in Endless Battle

New in version 0.11.1

  • The game is now available on Mac OS!
  • The game will now check for any instances of the game and quit if an instance of the game is already running before the game data is being loaded.
  • Fixed the bug that the window title is changed once you click the settings option.
  • Added shortcuts in settings.
  • You can now change your username in the settings dialog!
  • You can now press F1 during game play to check how many enemies are spawned.
