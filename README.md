# Endless Battle
This repository is for a game called **Endless Battle** and this is used to host mostly documentation and the web page. In addition, the repository's releases section includes the releases of the game and therefore you can easily download whatever the version and/or platform you wish to play, but it is recommended to play the most up to date version.

You can visit the [website](https://harrymkt.github.io/endless-battle) to read documentation and everything related to the game.
