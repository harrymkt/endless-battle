Shortcuts

General

  • R: Reloads the current weapon.
  • D: Brings up a menu to select weapon.
  • Control: Fires the currently selected weapon.

Movement

  • Spacebar: Jumps.
  • Arrow Keys, Page up and Page down: Moves around the map respectively.
  • Q: Turns 45 degrees left.
  • E: Turns 45 degrees right.

Informational

  • A: Checks nearby enemies.
  • F: Checks your direction.
  • H: Checks health.
  • Z: Checks information of the weapon ammo.
  • C: Checks coordinates.
  • Comma: Moves the buffer log previous.
    • Combine with shift key will move start.
  • Period: Moves the buffer log next.
    • Combine with shift key will move end.

Form

When the form is popped out, you can use the following keys.

  • Tab: Moves the control next.
    • Combine with shift key will move to previous.
    • Combine with control will move to the next tab control if there is available.
      • Combine with shift will move to the previous tab control if there is available.
  • Spacebar: Toggles options. For example, check and uncheck the checkboxes. Press if it is a button.
  • Enter: Performs action on a control. Press if it is a button.
  • Escape: Breaks the form.

List shortcuts

These shortcuts can be used if you are focusing on a list.

  • Spacebar: Check or uncheck the selectable item if it is multiselectable list.
  • Control+A: Selects all the items if it is a multiselectable list.
  • Up and Down arrows: Moves through the list items.
  • Page up and Page down: Moves the list items by the value depending on the set list, default is 5.
  • Home and End: Moves the focus to the start or the end of the list.

You can also press the letter keys to search items, using multilettter navigation system.
