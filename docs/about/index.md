# About

Endless Battle is an offline 3d FPS shooting game for the people with visual impairment, nothing but using a screen reader.

The game is plaied in a 3d environment world, where your task is to kill the enemies that spawned at every few seconds.

As you kill the enemies, you gain coins. These coins can be used to buy ammos and health related items in the store.

The game is developed by a man named Harry Min Khant, and was developed on a trusted scripting game engine that is publically available.
