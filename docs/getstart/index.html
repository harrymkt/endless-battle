Getting Started

Game objects

Menus

A menu is a containers that stores items to choose. The menu will update the items depending on what menu is selected. Main menu, for example, has the options play, settings, quit, ETC.

A menu can be navigated using your arrow keys. Specificly, this is the up and down arrow keys.

You can press enter to perform the selected item, and escape to break. Keep in mind that some menus can be disabled the escaping, meaning you won't be able to press escape key in that disabled menus.

In some possible ability, you can also press the tab key to repeat the title of the menu again, so you would know in case you don't hear the screen reader's speech when the menu is displaied.

Forms, Virtual dialogs

A form is a set of controls that displaies like a window, for instance, like windows checkboxes, lists, ETC.

Dialog, DLG

Dialog, or short for DLG, is a message dialog that displaies the message as alerting way. The dialogs themselves cannot be broken as not to lose the message. Instead, you can press arrow keys to read the message again, and escape or enter to break the dialog. Dialogs are useful to show most important notifications.

Print, alertbox

The alert box is kind of a dialog but with a different purpose. Once the alert dialog is popped out, there will be a button to break the dialog.

The button can different depending on the message box of the script. This alert box is useful to provide messages that are more or less important.

Another advantage on the alert box is that the message is shown in the device's display system, meaning other sighted people can be read and click the button with the mouse or the keyboard. Thus, this alert box does not use screen reader for support reading the message like other speech within the game.
