# A 3D Offline Shooting Game

Welcome to **Endless Battle**, a 3d, offline shooting game for the visually impaired using screen reader. The game is plaied using your keyboard, and there is no visuals.

To play the game it is necessary to have speaker that can hear from left and right, front and back, and up and down, as the game is completely in a 3d environment world.

# Requirements

* 1 GB ram, 2 GB is recommended. This is for the best possible performance.
* 50 MB disk space. The higher disk space you have, the better.
* Speakers with 3d environment support, possibly headphones.
* X64 system, this is required, the game will not run on 32 bit systems.
* Windows (8, 8.1, 10, 11); Linux (using Wine).
* For screen readers, the game is tested under NVDA, and SAPI5.
