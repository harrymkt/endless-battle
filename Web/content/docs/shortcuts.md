title="Shortcuts"
+++
* Q : Turns left 45 degrees.
* E : Turns right 45 degrees.
* R : Reloads.
* D : Brings up a menu to select a weapon to draw.
* F : Checks direction.
* H : Checks health.
* Either Control : Fires the selected weapon.
* Spacebar : Jumps.
* Arrow keys : Moves around the map.
* Escape : Exit to main menu.
